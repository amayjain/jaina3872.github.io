Lists element

Description

Lists help organize items in a set in a cleaner way to avoid cluster.

Syntax

<ol (or) ul>
<li>List item</li>
</ol (or) ul>

Example

<ol (or) ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol (or) ul>

Display

  1. Coffee
  2. Tea
  3. Milk


  • Coffee
  • Tea
  • Milk

Tables are an efficient way to sort data according to rows and columns.

Syntax

</tr>
<tr>

</tr>
</table>

Example

<table>
<tr>
<th>Firstname</th>
<th>Lastname</th>
<th>Age</th>
</tr>
<tr>
<td>Jill</td>
<td>Smith</td>
<td>50</td>
</tr>
<tr>
<td>Eve</td>
<td>Jackson</td>

</tr>
</table>

Eve Jackson 94
